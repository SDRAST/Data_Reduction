readHeaders¶

Data_Reduction.DSN.OLSR.readHeaders(f, options)¶

Decode the file header

This will read all the headers between startTime and endTime (or their defaults) and return them as dicts. If you want to see only the first header set option 'duration' to 1. If you want to see them on your screen also, set option 'toPrintHeaders' to 1.

Uses options ‘format’ and ‘toPrintHeaders’
