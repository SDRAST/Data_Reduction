get_last_line¶

Data_Reduction.DSN.get_last_line(fd)¶

Gets the last line in an ASCII file.

This is useful for getting the last line in a very large ASCII file. It also reports of the last line was completed with a newline or interrupted.

:param fd : file descriptor
