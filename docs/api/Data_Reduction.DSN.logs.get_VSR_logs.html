get_VSR_logs¶

Data_Reduction.DSN.logs.get_VSR_logs(year, DOY, dest_path)¶

Get logs from the VSR.

This gets the VSR logs, consisting of VSR* and vrt* logs in the VSR log directory and PESD logs in the ops home directory.

:param yearint

Year of observation

:param DOYint

Day of observation

:param dest_pathstring

Full path of directory where log copies should go.

Returns

tuple Names of VSR logs, VSR scripts, and VSR ‘at’ jobs.
