is_skinny¶

Data_Reduction.is_skinny(data_array, index)¶

Test whether a data value is an isolated outlier

Returns True if the data values adjacent to the test value are less that 1/10 of the test value, i.e., the data point is a spike

:param data_array : numpy array

:param index : int

Returns

boolean
