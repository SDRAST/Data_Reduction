eac_macro_file_name¶

Data_Reduction.DSN.logs.eac_macro_file_name(datadir, year, DOY)¶

Returns the name of the EAC macro log file name.

There should be only one in a observation day directory. If there are more it returns the last created one.

:param datadirstring

Observation date subdirectory full path

:param year : int

:param DOY : int

Returns

string EAC macro log file name
