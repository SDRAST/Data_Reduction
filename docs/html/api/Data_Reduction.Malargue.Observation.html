Observation¶

class Data_Reduction.Malargue.Observation(parent=None, name=None, dss=None, date=None, start=None, end=None, project=None)¶

Bases: Data_Reduction.DSN.Observation

Class for observations based on open-loop recording made as DSA-3 (DSS-84)

The arguments for the superclass initialization are::

parent (typically self or None), name (will be set to a YEAR/DOY default if not provided), dss (required), date (YEAR/DOY required), start (optional, usually taken from file header), end (otional, inferred from header and number of records), and project (required).
