get_obs_session¶

Data_Reduction.get_obs_session(project=None, dss=None, date=None, path='proj')¶

Provides project, station, year and DOY, asking as needed.

It follows one of several possible paths to get to the session:

proj - path through /usr/local/projects/project
hdf5 - path through /usr/local/RA_data/HDF5
fits - path through /usr/local/RA_data/FITS
wvsr - path through /data

:param project : optional name as defined in /usr/local/projects :type project : str

:param dss : optional station number :type dss : int

:param date : optional YYYY/DDD :type date : str

Returns

project, DSS, year, DOY.
