get_obs_session_old¶

Data_Reduction.get_obs_session_old(project=None, dss=None, date=None)¶

Asks user for parameters to locate observation session paths

This expects one of two directory trees to exist. If dtype is given::
/usr/local/RA_data/dtype/
dssXX/
YEAR/

DOY/

or if project is given::
/usr/local/projects/project/Observations/
dssXX/
YEAR/

DOY/

If neither is given it will prompt for a project.

:param project : optional name as defined in /usr/local/projects :type project : str

:param dss : optional station number :type dss : int

:param date : optional YYYY/DDD :type date : str

Returns

project, DSS, year, DOY.
