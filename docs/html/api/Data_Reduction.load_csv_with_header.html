load_csv_with_header¶

Data_Reduction.load_csv_with_header(filename, delimiter='')¶

Takes a text table with headers and converts it into an ASCII array

There must be a header for each column and the separator must not be ambiguous, for example, a space if the column headers also include spaces.

The data are ASCII and the program or programmer will have to convert them to an appropriate form, e.g. int or float or datetime, if necessary.

A column can be extracted using data[label].

:param filename : name of the data file :type filename : str

:param delimiter : column separator :type delimiter : str

Returns

column headers (str), data array(str)
